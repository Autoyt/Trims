### Configuration notes
- Borderlands meltdown duration is configured in ticks at `trims.world-options.duration.meltdown`.
  - 20 ticks = 1 second; 1200 ticks = 1 minute; 6000 ticks = 5 minutes.
  - Example: set `meltdown: 6000` for a 5-minute countdown.
